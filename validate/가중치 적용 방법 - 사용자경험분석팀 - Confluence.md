### 1\. 긍정 응답률

**정의**: 7점 척도에서 5~7점으로 응답한 사람의 비율

`예시: 전체 응답자: 1,000명 5~7점 응답자: 300명 긍정 응답률 = 300 / 1,000 × 100 = 30%`

### 2\. 집단별 가중치

**목적**: 표본의 집단 구성을 모집단과 일치시키기 위함

#### 1단계: 모집단을 MECE하게 분할

**분할 기준**: 연령대(3개) × 성별(2개) = 총 6개 집단

**모집단 구성 (실제 인구 비율)**

`20대 남성: 12% 20대 여성: 13% 30대 남성: 15% 30대 여성: 18% 40대 남성: 20% 40대 여성: 22% ───────────── 합계: 100% ✓`

**표본 구성 (실제 조사된 비율)**

`20대 남성: 8명 (8%) 20대 여성: 7명 (7%) 30대 남성: 25명 (25%) 30대 여성: 20명 (20%) 40대 남성: 15명 (15%) 40대 여성: 25명 (25%) ───────────────── 합계: 100명 (100%) ✓`

#### 2단계: 각 집단별 가중치 계산

**가중치 = 모집단 비율 ÷ 표본 비율**

`20대 남성: 12% ÷ 8% = 1.50 (과소표집 → 가중치 상향) 20대 여성: 13% ÷ 7% = 1.86 (과소표집 → 가중치 상향) 30대 남성: 15% ÷ 25% = 0.60 (과대표집 → 가중치 하향) 30대 여성: 18% ÷ 20% = 0.90 (과대표집 → 가중치 하향) 40대 남성: 20% ÷ 15% = 1.33 (과소표집 → 가중치 상향) 40대 여성: 22% ÷ 25% = 0.88 (과대표집 → 가중치 하향)`

#### 3단계: 각 집단의 응답 현황

`20대 남성: 8명 중 긍정 5명 (62.5% 긍정률) 20대 여성: 7명 중 긍정 5명 (71.4% 긍정률) 30대 남성: 25명 중 긍정 10명 (40.0% 긍정률) 30대 여성: 20명 중 긍정 12명 (60.0% 긍정률) 40대 남성: 15명 중 긍정 6명 (40.0% 긍정률) 40대 여성: 25명 중 긍정 8명 (32.0% 긍정률)`

#### 4단계: 가중치 적용 계산

**각 집단의 가중된 수:**

`20대 남성: 긍정 5×1.50=7.5,  전체 8×1.50=12.0 20대 여성: 긍정 5×1.86=9.3,  전체 7×1.86=13.0 30대 남성: 긍정 10×0.60=6.0,  전체 25×0.60=15.0 30대 여성: 긍정 12×0.90=10.8, 전체 20×0.90=18.0 40대 남성: 긍정 6×1.33=8.0,  전체 15×1.33=20.0 40대 여성: 긍정 8×0.88=7.0,  전체 25×0.88=22.0`

**전체 합산:**

`총 가중 긍정자 = 7.5+9.3+6.0+10.8+8.0+7.0 = 48.6 총 가중 전체자 = 12.0+13.0+15.0+18.0+20.0+22.0 = 100.0`

**최종 결과:**

`가중 응답률 = 48.6 / 100.0 × 100 = 48.6% 단순 응답률 = 46 / 100 × 100 = 46.0% 차이 = 48.6% - 46.0% = +2.6%p`

**해석**: 표본에서 긍정적 응답이 낮은 30대 남성/40대 여성을 과대표집했기 때문에, 가중치 보정 후 긍정 응답률이 상승함

### 3\. 가중 응답률 계산

**수식**:

`가중_응답률 = Σ(긍정응답자_가중치) / Σ(전체응답자_가중치) × 100`

**예시**:

`응답자 A (20대 여성 수도권, 긍정, 가중치 1.5) 응답자 B (30대 남성 지방, 부정, 가중치 2.0) 응답자 C (20대 여성 수도권, 긍정, 가중치 1.5) 긍정 응답자 가중치 합 = 1.5 + 1.5 = 3.0 전체 응답자 가중치 합 = 1.5 + 2.0 + 1.5 = 5.0 가중 긍정 응답률 = 3.0 / 5.0 × 100 = 60%`

### 4\. 표본 편향

**표본이 모집단과 다른 구성을 가질 때 발생**

**편향의 방향**:  
\- **긍정적 편향**: 실제보다 긍정적 응답자가 많음 → 가중치 적용 시 응답률 하락  
\- **부정적 편향**: 실제보다 부정적 응답자가 많음 → 가중치 적용 시 응답률 상승
